# 垂直视频布局更新

## 更新内容

### 1. VideoCard 组件重构
- **纵横比调整**: 从 `aspect-video` (16:9) 改为 `aspect-[9/16]` (9:16) 以适配垂直视频
- **移除 placeholder 缩略图**: 不再使用 placeholder 图片，直接使用视频的第一帧作为缩略图
- **简化交互**: 移除了复杂的缩略图/视频切换逻辑，改为点击打开模态框播放

### 2. 视频数据结构简化
- 移除了 `thumbnail` 属性
- 保留 `id`, `title`, `description`, `video` 属性

### 3. 布局优化
- **垂直布局**: 视频卡片现在使用 9:16 的纵横比
- **悬停效果**: 添加了缩放和位移动画效果
- **渐变叠加**: 在视频底部添加渐变叠加，确保文字可读性
- **模态框播放**: 点击视频卡片在模态框中全屏播放视频

### 4. 视觉改进
- **播放按钮**: 居中显示的大型播放按钮
- **标题和描述**: 显示在视频底部的渐变叠加上
- **响应式设计**: 在不同屏幕尺寸下保持良好的显示效果

## 技术实现

### VideoCard 组件特性
```typescript
- 使用 aspect-[9/16] 适配垂直视频
- 直接使用 <video> 元素显示视频第一帧
- preload="metadata" 加载视频元数据
- AnimatePresence 实现模态框动画
- 点击外部区域关闭模态框
```

### 视频数据格式
```typescript
{
  id: number,
  title: string,
  description: string,
  video: string  // 视频文件路径
}
```

## 用户体验
- 垂直视频格式更适合移动设备观看
- 直接显示视频内容而非占位图
- 流畅的动画和交互效果
- 一键播放，简单直观

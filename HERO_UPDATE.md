## 🔧 技术实现

### 当前代码结构：
```typescript
// 简化的 Hero Section
<img 
  src="/images/hero1.PNG" 
  alt="KK Cheese Hero" 
  className="relative z-10 w-full max-w-[400px] mx-auto drop-shadow-2xl hover:scale-105 transition-transform cursor-pointer"
/>
```

### 保留的动画效果：
1. **进入动画**：`scale: 0.8 → 1`, `rotate: -5° → 0°`
2. **悬停效果**：`hover:scale-105` (5% 放大)
3. **浮动徽章**：`y: [0, -10, 0]` 循环动画
4. **背景光晕**：`animate-pulse` CSS 动画

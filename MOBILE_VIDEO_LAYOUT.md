# 移动端视频布局优化

## 更新内容

### 1. 网格布局调整
- **移动端**: 从 `grid-cols-1` 改为 `grid-cols-2` (2x2布局)
- **平板端**: 保持 `md:grid-cols-2` 
- **桌面端**: 保持 `lg:grid-cols-4`
- **间距优化**: 移动端使用 `gap-3`，桌面端使用 `md:gap-6`

### 2. VideoCard 组件优化

#### 视频卡片尺寸
- **圆角**: 移动端 `rounded-xl`，桌面端 `md:rounded-2xl`
- **纵横比**: 保持 `aspect-[9/16]` 适配垂直视频

#### 播放按钮响应式
- **移动端**: `w-10 h-10` (40x40px)
- **桌面端**: `md:w-16 md:h-16` (64x64px)
- **图标大小**: 移动端 `size={20}`，桌面端 `md:w-8 md:h-8`

#### 文字内容响应式
- **内边距**: 移动端 `p-2`，桌面端 `md:p-4`
- **标题**: 移动端 `text-xs`，桌面端 `md:text-lg`
- **描述**: 移动端 `text-xs`，桌面端 `md:text-sm`

### 3. 用户体验改进
- **紧凑布局**: 移动端2x2网格节省屏幕空间
- **点击播放**: 点击小卡片后在模态框中全屏播放
- **渐进增强**: 从移动端小尺寸到桌面端大尺寸的平滑过渡

## 布局对比

### 移动端 (< 768px)
```
┌─────┬─────┐
│ 视频1 │ 视频2 │
├─────┼─────┤
│ 视频3 │ 视频4 │
└─────┴─────┘
```

### 桌面端 (≥ 1024px)
```
┌─────┬─────┬─────┬─────┐
│ 视频1 │ 视频2 │ 视频3 │ 视频4 │
└─────┴─────┴─────┴─────┘
```

## 技术实现

### CSS 类名更新
```css
/* 网格布局 */
grid-cols-2 md:grid-cols-2 lg:grid-cols-4 gap-3 md:gap-6

/* 播放按钮 */
w-10 h-10 md:w-16 md:h-16

/* 文字大小 */
text-xs md:text-lg  /* 标题 */
text-xs md:text-sm  /* 描述 */

/* 内边距 */
p-2 md:p-4
```

## 优势
1. **移动优先**: 在小屏幕上提供更好的浏览体验
2. **空间利用**: 2x2布局充分利用移动端屏幕空间
3. **交互友好**: 小卡片易于点击，大模态框易于观看
4. **性能优化**: 较小的缩略图减少初始加载时间

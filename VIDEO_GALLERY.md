## 🎯 用户体验流程

### 1. 初始状态
- 用户看到4个彩色缩略图
- 每个缩略图中央有播放按钮
- 鼠标悬停时按钮放大

### 2. 点击播放
- 缩略图切换为实际视频
- 视频自动开始播放
- 显示原生视频控件

### 3. 视频控制
- 点击视频可播放/暂停
- 使用原生控件调节音量、进度
- 视频结束后自动暂停

### 4. 返回缩略图
- 点击 "Back to thumbnail" 链接
- 视频重置到开始位置
- 回到缩略图模式

## 🔧 自定义配置

### 如果需要更换视频：
1. 将新视频文件放入 `images/` 文件夹
2. 更新 `App.tsx` 中的视频配置：

```typescript
{
  id: 1,
  title: "新视频标题",
  description: "新视频描述", 
  video: "/images/new-video.mp4",
  thumbnail: "新缩略图URL"
}
```

### 如果需要调整缩略图：
- 可以使用实际视频截图替代占位图
- 或者使用设计工具创建自定义缩略图
